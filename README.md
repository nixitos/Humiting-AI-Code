# Humiting AI Code
Этот инструмент позволяет очеловечивать любой код от LLM через инструкции, которые были выведены из просмотра человеческих скриптов
Данный инструмент конкретно меняет логику ответов от LLM так что используйте её в новых чатах

---

# Как использовать?
Легче некуда, просто отправьте LLM первое сообщение как из примера:
```
https://raw.githubusercontent.com/nixitos/Humiting-AI-Code/refs/heads/main/main-prompt.txt
<Ваш код>
```
*То есть, сначала обязательная ссылка ведущая на промпт. На следующей строке ваш код*


Этот промпт будет работать только на тех LLM, которые поддерживают чтение контента из ссылок (ChatGPT, DeepSeek, Grok)
